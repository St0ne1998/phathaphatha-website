Phathaphatha Solutions Holdings Website

A modern, premium-level business website built for Phathaphatha Solutions Holdings (Pty) Ltd, a multi-service company based in South Africa.

------------------------------------------------------------

OVERVIEW

This website is a single-page application (SPA) built using HTML, CSS, and JavaScript. It provides a smooth, interactive, and professional user experience.

------------------------------------------------------------

FEATURES

- Premium modern UI design
- Fully responsive (mobile, tablet, desktop)
- Smooth scrolling navigation
- Custom animated cursor
- Scroll progress indicator
- Interactive project slideshow/gallery
- Scroll reveal animations

Sections included:
- Hero
- About
- Services
- Safety
- SWOT Analysis
- Contact Form

------------------------------------------------------------

TECHNOLOGIES USED

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts
- Responsive Design (Media Queries)

------------------------------------------------------------

PROJECT STRUCTURE

project-folder/
│── phathhaphatha.html   (Main website file)
│── images/              (Local images if used)
│── README.txt           (Project documentation)

------------------------------------------------------------

GETTING STARTED

1. Download or clone the repository
2. Open the file "phathhaphatha.html" in any web browser

------------------------------------------------------------

CUSTOMIZATION

You can edit:
- Text content in the HTML file
- Colors in the CSS :root variables
- Images (replace with your own)
- Contact details in the contact section

------------------------------------------------------------

NOTES

- This is a front-end only project
- The contact form is for UI purposes only (no backend functionality)
